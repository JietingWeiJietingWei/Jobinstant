# ⚡ JD Fit Analyzer — Chrome Extension
## Install in 3 steps (no app store needed)

### Step 1 — Open Chrome Extensions
1. Open Chrome and go to: `chrome://extensions`
2. Toggle **Developer mode** ON (top-right switch)

### Step 2 — Load the extension
1. Click **"Load unpacked"** button (top-left)
2. Select the **`jd-analyzer-extension`** folder (this folder)
3. The extension icon ⚡ appears in your Chrome toolbar

### Step 3 — Configure your resume + API key
1. Click the ⚡ icon in your toolbar
2. Paste your **resume text** in the box
3. Enter your **Anthropic API key** (get one free at console.anthropic.com)
4. Click **Save Settings**

### Done! Go to LinkedIn
- Navigate to any job on `linkedin.com/jobs`
- A score badge appears automatically below the job title
- Click the badge to expand the full analysis

---
## Getting an Anthropic API Key
1. Go to https://console.anthropic.com
2. Sign up (free) → go to "API Keys"
3. Click "Create Key" → copy it
4. Paste into the extension popup

Each analysis uses ~600 tokens (~$0.002 per job viewed)

---
## Notes
- Your resume and API key are stored **locally on your device only**
- The extension only runs on `linkedin.com/jobs/*` pages
- Score updates automatically when you switch between jobs
