JobInstant求职插件安装说明
这是什么？
JobInstant	是一个	Chrome	插件，帮你在	LinkedIn	上	一键打分职位、看清自己的匹配度、自动优化简历。再也不用花几个小时纠结「我到底适不适合这个工作」了。
安装环境：
JobInstant插件需要在电脑端，Chrome浏览器上用 

安装步骤：
1.	下载安装包
https://github.com/JietingWeiJietingWei/Jobinstant/archive/refs/heads/main.zip

 
2.	解压文件夹到本地

3. 打开 Chrome	扩展页面
在	Chrome	浏览器的地址栏粘贴这个网址：
chrome://extensions
然后按	Enter	键。

4. 打开「开发者模式」
在扩展页面的右上角，找到「开发者模式」开关，点一下把它打开（开关应该变成蓝色）。
 
5. 加载解压后的文件夹
开启开发者模式后，页面左上角会出现	「加载已解压的扩展程序」	按钮。点一下它。
 

6. 选择	JobInstant	文件夹
浏览器会弹出一个文件夹选择框。找到你解压的	JobInstant	文件夹，选中它，点	「选择文件夹」。

7. 确认安装成功
如果一切顺利，扩展页面上会出现一个JobInstant的卡片，说明安装成功了！

8. 固定插件（可选但推荐）
点	Chrome	右上角的	扩展程序图标，在	JobInstant	旁边的菜单里选	「固定」，这样	JobInstant	的图标就会一直显示在工具栏里。
 
首次使用：上传简历
第一次用的时候，需要上传你的简历。JobInstant	用你的简历来分析你和职位有多匹配。
1. 点	Chrome	工具栏里的JobInstant图标
2. 在弹出的窗口里上传简历
3. 上传后自动保存！
4.  现在可以去LinkedIn试试了

怎么用	JobInstant
打开LinkedIn	职位页面（去linkedin.com/jobs）

1.	Score Jobs:
页面上会有一个蓝色的「Score	Jobs」按钮，点一下
列表右侧就会批量出现你的匹配分数
点击左侧职位，右边边栏会弹出来 

2.	右侧边栏详情：
	必须要有的需求（Must-have Fit）
	技能和经验匹配度（Skills	& Experience	Fit）
	加分项（Nice-to-have	Fit）
每一项都有一个百分比，一眼就能看出你的优势和不足决定要不要投
根据拆解，你就知道：要投还是跳过。如果匹配度高，继续下一步

3.自动优化简历
点右边栏底部的「Rewrite	Resume」按钮，
JobInstant	会根据这个职位的需求自动改你的简历，把你最相关的经历突出出来。改好后，你就可以直接投了
